### 任务格式说明

| 格式 | 含义 |
|------|------|
| `- [ ]` | 未完成任务 |
| `- [x]` | 已完成任务 |
| `📅 日期` | 截止日期（Tasks 插件识别） |
| `🏷️ #tag` | 标签（Tasks 插件识别） |
| `#high` | 高优先级标签 |
| `#medium` | 中优先级标签 |
| `#low` | 低优先级标签 |
